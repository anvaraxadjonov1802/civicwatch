# CivicWatch — InTash Hackathon Project

## About the Project

**CivicWatch** — bu maktablar infratuzilmasi, holati va foydalanuvchi hisobotlarini kuzatish uchun yaratilgan zamonaviy civic-tech platforma.

Loyiha foydalanuvchilarga:
- maktablarni qidirish,
- ular bo‘yicha holatni ko‘rish,
- muammo yoki bajarilgan ish haqida hisobot yuborish,
- yuborilgan arizalar holatini kuzatish

imkonini beradi.

Platforma ayniqsa:
- ota-onalar,
- o‘qituvchilar,
- mahalla faollari,
- ta’lim muassasalari bilan qiziqadigan foydalanuvchilar

uchun qulay va tushunarli tarzda ishlab chiqilgan.

## Main Features

### User Features
- Maktablarni qidirish va ko‘rish
- Maktablar bo‘yicha umumiy ma’lumotlarni ko‘rish
- Muammo yoki bajarilgan ish haqida hisobot yuborish
- Qurilmadan rasm yuklash yoki kamera orqali rasmga olish
- Yuborilgan arizalar uchun tracking number olish
- Ariza holatini kuzatish
- Holat tarixi (timeline) ni ko‘rish

### Platform Features
- Zamonaviy va mobilga mos UI
- 3 tilli ishlash:
  - O‘zbek
  - Rus
  - Ingliz
- Light / Dark mode
- Gamification elementlari:
  - badge
  - points
  - reward cards
- Demo uchun statuslarni qo‘lda boshqarish imkoniyati

## Problem Statement

Ko‘plab hollarda maktablardagi muammolar haqida foydalanuvchilar xabar beradi, lekin keyinchalik:
- bu xabar ko‘rib chiqildimi,
- muammo hal qilindimi,
- ariza qaysi bosqichda

degan savollarga javob topa olmaydi.

**CivicWatch** shu muammoni hal qilish uchun:
- hisobot yuborish,
- kuzatish,
- statusni ko‘rish,
- natijalarni shaffof tarzda taqdim etish

imkonini beradi.
